# Generic Sandbox

This is a generic/public sandbox environment for testing and experiments.

## Restrictions

- Read-only access
- No sensitive data
- Limited file types (.txt, .md, .json, .log, .csv)
- Max 2 levels deep
- No execution permissions
- Isolated from other projects

## Usage

This space is used as a fallback when:
- No specific project ID is provided
- Project context cannot be determined
- Testing bot functionality
- Public API demonstrations

## Access

- **Business Unit ID**: 999
- **Project ID**: 999
- **Status**: Active (Development Environment)
- **Intelligence Level**: Basic
